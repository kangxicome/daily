### üì° OpenWrt

1. Put the [`scripts`](openwrt) into OpenWrt.
2. Add the following cron jobs.

```bash
# adblock
0 3 * * * /path/to/adblock.sh

# chinadns-ng
1 3 * * * /path/to/chinadns-ng.sh

# chinadns
2 3 * * * /path/to/chinadns.sh
```
